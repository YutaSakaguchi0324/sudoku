# プログラム概要
gurobipyを用いて数独を解くプログラムをつくる

# 実行環境
windows 10 64bit

Anaconda 4.13.0

gurobipy 9.5.1

# プログラムの使い方
変数sが数独の問題設定です。.(ドット)は空欄を示しています。
.(ドット)や数字を自由に書き換えて実行することで、その問題の解をnumpy.ndarrayで出力します。
